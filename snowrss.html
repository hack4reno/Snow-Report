SnowRSS

An RSS Extension for Snow Conditions Reports

What is SnowRSS

SnowRSS is an extension to RSS v2 that defines new elements and entities to describe snow, lift and weather conditions for ski resorts.

Publishing snow conditions in SnowRSS allows ski resorts to get their conditions data out to developers and skiers in a format that's easy to use and build on.

Channel Element Additions

SnowRSS adds several elements to the standard channel section.

Element Required Description Example
resortName • The name of this ski resort
conditionsLink • The URL to your HTML Conditions page
Status • Indicates whether the resort is open. Must be, "open" or "closed"
scheduledOpenDate The date the resort is scheduled to open.
Temperature Current temperature
Winds Wind speed and direction
weatherConditions Description of the current weather
Visibility Visibility distance
Forecast Weather forecast
baseDepth Snow depth at the base
newSnow24 Depth of new snow fallen in the last 24 hours.
newSnow48 Depth of new snow fallen in the last 24 hours.
stormSnowTotal Total snow fall from the most recent storm.
seasonSnowTotal Total snow fall for the season

Item Element Additions

SnowRSS extends the Item tag with new several entities and elements to describe the status of lifts and runs.

Additional Entities of Item

snowRSSType Each Item element must specify a snowRSSType entity of "lift" or "run" to define the entity type. Resort features such as terrain parks should be labeled "run".

Additional Elements of Item

Lifts

Title The Title element of a lift Item should be used for the lift name.
Description The Title element of each Item should be a free form text description of the lift status
Status Indicate whether the lift is open. Status must be "open" or "closed".
liftType Describe the list type, tow rope, gondola, etc.

Runs

Title The Title element of a lift Item should be used for the run name.
Description The Title element of each Item should be a free form text description of the run status
Status Indicate whether the run is open. Status must be "open" or "closed".
Conditions Describe the snow conditions on the run, "Powder", "Groomed Snow", "Brush and Rocks", etc.
Difficulty Indicates the run's standard difficulty level, and should be on of, "Beginner", "Intermediate", "Advanced" or "Foolhardy".
